Element Example Explination
div <div> </div> div can be used as a container for other elements
p <p>text</p> p is used as a paragraph
h(number) <h1>text</h1> h(number) is a header. h1 should only be used once per page and go lower to help segment the website
button < button onclick="function('input')">text</button> button which on click can call a function the ' are needed in order to have a proper input of a variable
ol <ol></ol> base for an ordered list
ul <ul></ul> base for an unordered list
li <li>something</li> the item part of a list
table <table></table> base for a table
tr <tr></tr> column of a table
td <td></td> row of a table
